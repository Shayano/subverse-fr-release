# Subverse - Patch de Traduction Française (v1.2.01)

Traduction française complète pour **Subverse** v1.2.01 (StudioFOW).
**10 799 chaînes traduites** — menus, dialogues, descriptions, codex, combats.

## Installation

1. Télécharger `Subverse_FR_v1.2.01.7z` depuis la page [Releases](../../releases)
2. Extraire l'archive **dans le dossier du jeu** :
   ```
   Steam\steamapps\common\Subverse\
   ```
   L'arborescence `Subverse\Content\Paks\` se crée automatiquement.
3. Lancer le jeu

> Chemin final du fichier :
> `Steam\steamapps\common\Subverse\Subverse\Content\Paks\pakchunk0_s99-WindowsNoEditor_French.pak`

### Désinstallation

Supprimer `pakchunk0_s99-WindowsNoEditor_French.pak` du dossier `Paks`.

### Limitations connues

Certains éléments du jeu sont intégrés directement dans le code ou les assets du moteur et ne passent pas par le système de localisation. Ils s'afficheront **en anglais** :

- Quelques labels d'interface (placement de personnages en combat, certains boutons)
- Noms des personnages dans certains écrans spécifiques
- Éléments des mini-jeux

Cela n'affecte pas les dialogues, menus principaux ni la compréhension du jeu.
